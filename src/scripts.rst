AMIEMediator Scripts
====================

The **AMIEMediator** package includes three python scripts.

The main script is ``amie``, which implements the mediator daemon program.

The ``viewpackets`` script is a command-line tools for viewing the state of
all requests.

The ``test-scenario`` script sets up an AMIE test scenario.
